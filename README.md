# Bootstrap_Course

## Running locally

1. Run ```npm install``` to install the Node.js dependencies
2. Run ```npm start``` to start the application


## Components included

* Grid System
* Responsive Design
* Navigation, Navigation Bar
* Buttons, Forms, Tables
* Cards, Media, Tags
* Tabs
* Accordian
* Modals
* Corousel


## Building and Deployment

* NPM Scripts
* Onchange
* Parallelshell
* Task Runners
* Grunt
* Gulp
